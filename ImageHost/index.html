ChapterCore: TSA Chapter Chat/Archive

For the 2024-2025 TSA Software Development Project

Welcome to ChapterCore!

Your one-stop solution for managing TSA chapter communications and archives.

Main Features:

  • Announcements: Receive announcements with push notifications, SMS, and email.
  • Event Rooms: Create separate rooms for each event, assignable manually or via CSV, and archived yearly.
  • Member Communication: Chat with other members within the app.
  • Archive: Store all code, designs, and portfolios securely.
  • API Access: Utilize our API in other applications with rate limiting by Cloudflare.
  • Gantt Charts & Audit Logs: Manage and track project progress with built-in Gantt charts and detailed audit logs.

Additional Information:

Join Policy:

Members will need authentication managed by the president, vice president, sergeant-at-arms, and chapter advisor. All account information is secure and encrypted.

Parental Usage:

Parents can sign up for announcement notifications and create direct messages with any user but cannot access archives or chats except for announcements.

Archive Policy:

  • Messages saved up to 5 years unless specified.
  • Event archives with specs, user guides, portfolios, and documentation are stored.
  • Events without a theme are hidden from users except those who worked on them.

Charts:

  • Gantt Charts: Auto-populated with user-specific colors and task statuses.
  • List Views: Task management with priority and due date columns.
  • Board Views: Visual task tracking with To-Do, In-Progress, and Done categories.

Roles:

  • Admins: Manage rooms, messages, and access tokens; control visibility of event archives.
  • Chapter Advisors: Add/remove rooms/events, manage admins, and have all admin capabilities.
  • Project Managers: View and manage all project-related content.
